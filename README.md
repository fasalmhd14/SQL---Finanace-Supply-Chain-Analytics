GUIDED PROJECT: This Project is part of the SQL Module in the Data Analysis Bootcamp by Codebasics.

- JOINS
- SUBQUERIES

- AGGREGATE FUNCTIONS, GROUP BY & HAVING
- SORTING
- ORDER OF EXECUTION IN SQL
- USER DEFINED FUNCTIONS
- STORED PROCEDURES
- VIEWS
- Creating New Tables
- WINDOW FUNCTIONS
